# togo
ToDo list application made by golang

## Usage
#### データベースの初期化
```
$ togo init
```
#### タスクの追加
```
$ togo add task1
Added new task : task1

$ togo add task2
Added new task : task2

$ togo list
+----+-------+--------+---------------------+
| NO | TITLE | STATUS |       CREATED       |
+----+-------+--------+---------------------+
|  1 | task1 | -      | 2021-10-12 16:25:11 |
|  2 | task2 | -      | 2021-10-12 16:25:14 |
+----+-------+--------+---------------------+
```
#### タスクの完了
```
$ togo done 1
Task 1 is done

$ togo list -a
+----+-------+--------+---------------------+
| NO | TITLE | STATUS |       CREATED       |
+----+-------+--------+---------------------+
|  1 | task1 | Done   | 2021-10-12 16:20:56 |
|  2 | task2 | -      | 2021-10-12 16:25:14 |
+----+-------+--------+---------------------+
```
#### タスクの削除
```
$ togo delete 1
Deleted task id = 1

$ togo list
+----+-------+--------+---------------------+
| NO | TITLE | STATUS |       CREATED       |
+----+-------+--------+---------------------+
|  2 | task2 | -      | 2021-10-12 16:25:14 |
+----+-------+--------+---------------------+
```
